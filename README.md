# Efood - Aplicação Front-End com React

## Sobre o projeto
Aplicação front-end inspirada em plataformas de delivery, desenvolvida com React. O projeto simula um fluxo de navegação e exibição de produtos utilizando consumo de API, com foco em experiência do usuário e organização de código.

## Tecnologias utilizadas
- React
- JavaScript
- CSS
- React Router
- API REST (fake)

## Funcionalidades
- Listagem de restaurantes/produtos
- Navegação entre páginas
- Exibição dinâmica de dados via API
- Interface responsiva
- Simulação de fluxo de navegação do usuário
